# In The Closet
### 안드로이드 스마트폰을 이용한 Virtual Fitting(가상 피팅) 애플리케이션  



<br>


# 📰 프로젝트 개요

**어디서나 옷을 피팅해보세요!**
<br> <br>
언택트 소비가 급증하면서 특히 옷 쇼핑에 있어서 피팅을 하지 못해 불편함을 느끼고 있습니다. <br> 자신의 스마트폰으로 어디서나 옷을 피팅해보고 룩북에 저장하여 보세요.
